### Fattoriale non ricorsivo
```c
int factorial(int n){  
    int result = 1;  
    for (int i = 1; i <= n; ++i) {  
        result = result * i;  
    }  
    return result;  
}

int main() {  
    factorial(5);  
    return 0;  
}
```

### Fattoriale ricorsivo
```c
int factorialR(int n){  
    if (n <= 1){  
        return n;  
    }  
    return n * FactorialR(n-1);  
}  
  
int main() {  
    factorialR(5);  
    return 0;  
}
```

Le funzioni ricorsive sono molto lente e occupano molto stack essendo che ogni chiamata di funzione usa molta memoria perchÃ¨ lo stack deve salvare l'indirizzo di ritorno e passare i parametri.
